## 🚀 Key Features

The project focuses on delivering maximum user experience (UX) and utilizes modern web technologies.

| Feature | Description |
| :--- | :--- |
| 🔐 **Smart Authentication** | Registration and login via Email/Password or Google account. Includes Enter key support for login and protection against intrusive browser autofill. |
| ☁️ **Cloud Database** | Integrated with Supabase. Tasks are linked to your account, synced in real-time, and persist seamlessly across all your devices. |
| 📱 **PWA (Installable App)** | The app can be installed directly to your Home screen (iOS/Android) or PC desktop. Features adaptive logic for displaying tailored installation and uninstallation instructions. |
| 🎨 **Themes & Glassmorphism** | Premium dark design with frosted glass effects. Includes a floating button for instant switching between light, dark, and system themes. |
| 🛡 **Secure Logout** | Implemented a `Double-click to confirm` system for the logout button to prevent accidental logouts, especially on mobile devices. |
| 🔍 **Task Filtering** | Instant switching between all, active, and completed tasks with automatic statistics calculation. |

## 💡 Interface Features (UX/UI)
* **Responsiveness:** The interface adapts perfectly to any screen size — from iPhones to wide desktop monitors.
* **Subtle Placeholders:** Input placeholders are semi-transparent so they don't distract attention, creating a proper visual hierarchy.
* **Context Retention:** When switching between the "Login" and "Register" tabs, the entered email is saved to prevent re-typing.
* **Animations:** Smooth task appearance, a stylish loading screen, and beautiful hover effects for all interactive elements.
